Método Prompt()


O método prompt() faz com que uma mensagem seja exibida para o usuário e o processamento do software pare, aguardando uma resposta do usuário.
Veja um exemplo:

prompt(“Digite um número”);

Aqui a frase “Digite um número” será exibida e o programa vai aguardar uma interação do usuário.
Mas assim que o usuário digitar algo e pressionar Enter, precisamos armazenar isso em algum lugar. Senão o programa “perde” o que foi digitado.

Então precisamos guardar o que foi informado em uma variável, utilizando um operador de atribuição, assim:

var primeiro = prompt(“Digite um número”);

Agora, o que o usuário digitar será armazenado na variável com o nome primeiro.

O que e um Number em JavaScript?


O objeto JavaScript Number é um objeto encapsulado que permite você trabalhar com valores numéricos. Um objeto Number é criado utilizando o construtor Number().

Formatando Strings

var s = 'JavaScript'
'Estou aprendendo s' \\Não fez interpolaçao.
'Estou aprendendo' + s \\Usou concatenaçao.
´Estou aprendendo ${s}´ \\Usou template string.

Algumas dicas de formatar strings

s.length //quantos caracteres a string tem

s.toUpperCase() //tudo para maiúsculas

s.LowerCase() //tudo para minúsculas
